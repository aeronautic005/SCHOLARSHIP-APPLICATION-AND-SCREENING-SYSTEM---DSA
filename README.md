# SCHOLARSHIP-APPLICATION-AND-SCREENING-SYSTEM---DSA
The Scholarship Management System is a console-based application developed in C++ designed to efficiently manage scholarship applications in an organized and structured way. It allows users to add, view, search, and delete applicant records, evaluate eligibility based on GPA and document completion, generate performance reports, and rank applicants according to their average scores. The system also includes a login and sign-up feature with file handling to ensure user accounts are stored and retained even after the program is closed. By utilizing data structures such as structs, vectors, and maps, the program ensures efficient data organization, fast processing, and easy scalability. This project demonstrates how fundamental programming concepts can be applied to create a functional and practical real-world system.
